# ! PDP7 Unix

![! PDP7 Unix](./pdp7unix.webp)

! PDP7 Unix was an operating system for the DEC PDP-7 computer. It is the earliest available Unix version to be able to run in an emulator. We can run it on the SIMH PDP-7 emulator.

## Manuals
