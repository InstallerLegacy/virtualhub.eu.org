# ! HP 2100 DOS-M

import DocCardList from '@theme/DocCardList';

![! HP 2100 DOS-M](./hp2100dos-m.webp)

! HP 2100 DOS-M, as the name suggests, was an operating system for the HP 2100 computer. We can run using the SIMH emulator.
