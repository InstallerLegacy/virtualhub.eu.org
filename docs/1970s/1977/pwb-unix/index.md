# ! PWB Unix

![! PWB Unix](./pwb-unix.webp)

[! PWB Unix](https://en.wikipedia.org/wiki/PWB/UNIX) was an operating system for the DEC PDP-11 computer. It was developed at Bell Labs. It can be used on SIMH PDP-11 emulator.

<DocCardList />
