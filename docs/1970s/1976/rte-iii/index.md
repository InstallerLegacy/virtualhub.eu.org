# ! HP 2100 RTE-III

import DocCardList from '@theme/DocCardList';

![! HP 2100 RTE-III](./rte-iii.webp)

! HP 2100 RTE-III, as the name suggests, was an operating system for the HP 2100 computer. We can run using the SIMH emulator.

<DocCardList />
